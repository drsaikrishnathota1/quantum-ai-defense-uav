# Services

This folder is reserved for modular service components used by the research pipeline.

## Planned services

### 1. ai-threat-service
Responsible for:
- threat scoring,
- hostile UAV prioritization,
- future ML model integration.

### 2. quantum-optimizer-service
Responsible for:
- defender-to-threat assignment,
- QUBO / quantum-inspired optimization experiments,
- future QAOA or annealing integration.

### 3. auth-security-service
Responsible for:
- crypto-agile authorization,
- command validation,
- post-quantum-ready secure control abstractions.

## Current state

At the moment, the repository uses a single-script prototype in `sim/main.py`.
This folder documents the intended modular architecture for future expansion.
